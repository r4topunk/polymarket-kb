# AARO

All-domain Anomaly Resolution Office — Pentagon office handling UAP/UFO investigations and classified material disclosure.

## Key events
- AARO responsible for classified UAP file releases ordered via Trump executive order

## Profile
Established 2023 as successor to UAPTF. Consolidated DoD, intelligence community, and civilian agency UAP investigations under unified command structure.
